**URL Shortener Service**

**A scalable URL Shortener backend built with Node.js, Express, MongoDB, and Mongoose.
The service generates short URLs, tracks total clicks on redirect URL, records visit timestamps, and provides **analytics** for each shortened link.**

**Features**
1. Generate short URLs from long URLs
2. Unique shortId generation for every URL
3. Redirect short URLs to the original destination
4. Track total number of clicks per short URL
5. Store visit timestamps for every redirection
6. Analytics endpoint for click history
7. Clean REST API structure

**Tech Stack**
- Backend: Node.js, Express.js
- Database: MongoDB, Mongoose
- ID Generation: shortid
- Dev Tools: Nodemon
- Deployment Ready in Future: Render (Backend), MongoDB Atlas (Database)
